MathPanel - набор приложений для компьютерного моделирования и визуализации, а также обучения программированию, в частности на языках C# и JavaScript. Инструмент разработки – MS Visual Studio 2019/2022 Community Edition или VS Code.

На базе проекта написана книга 
Борзяк А. А., Топорков В. В., Емельянов Д. М., Самочёрнов О. И., Смирнов Р. С. Основы компьютерного моделирования и визуализации. Издательство «Лань», 2022 год, 244 с., ISBN 978-5-507-44951-4

Для контактов с авторами используйте почту «Борзяк Андрей Александрович» aborziak@yandex.ru .


Подготовка к работе.
В конфигурационном файле MathPanel.app надо поправить строку "netframworkpath", если Вы используете другую версию фреймворка.
    <!--path to framework -->
    <add key="netframworkpath" value="C:\Program Files (x86)\Reference Assemblies\Microsoft\Framework\.NETFramework\v4.8\" />

Для компиляции, возможно, потребуется пакет Npgsql.4.1.3.1 и ряд других (установить через NuGet).

После компиляции в папки Bin/Debug и Bin/Release скопировать
graphix.js, jquery-1.11.1.min.js, test_graph.htm, ru.local, scripts/template.cs

Перед первым запуском MathPanel.exe надо выполнить "2.reg" для добавления ключей в регистр.
При переходе по кнопке "График" возможно потребуется разблокировать контент веб-страницы.

Вложенные папки.
Geom, MathExt - код гометрических объектов и дополнительных классов.
Book - документы с кратким описанием MathPanel
pictures - тестовые скрипты для двумерного рисования.
scipts - тестовые скрипты для моделирования и визуализации.
Пример рисования текста и линий вместе с геометрическими объектами дан в "scripts\test36_text_3d.cs".

EnergyGen - решение для поиска цены на распределение электроэнергии.
MeterForm - решение для демонстрации клиент-серверного взаимодействия.
MathPanelCore_net8 - решение для Linux на базе .Net Core 8.
MathPanelCore - решение для Linux на базе .Net Core 3.1.
